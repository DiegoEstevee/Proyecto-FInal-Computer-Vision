# Proyecto Final – Computer Vision

Proyecto de Visión por Ordenador en tiempo real con OpenCV que integra dos módulos independientes:

1. **Sistema de seguridad mediante contraseña visual**, basado en detección de letras.
2. **Sistema de análisis de ping-pong**, con detección y seguimiento de la pelota, detección de botes y cálculo automático de puntos.

Todos los resultados se guardan automáticamente en la carpeta `results/`.
